string formatDate( array $params, null $template = null)

return date in expected format

available parameters : date => DateTime object (mandatory) format => expected format output => list of default system format. Values available : date => date format time => time format datetime => datetime format (default)

ex : {formatdate date=$dateTimeObject format="Y-m-d H:i:s"} will output the format with specific format {formatdate date=$dateTimeObject format="%e %B %Y" locale="frFR"} will output the format with specific format (see strftime() function) {formatdate date=$dateTimeObject output="date"} will output the date using the default date system format {format_date date=$dateTimeObject} will output with the default datetime system format
