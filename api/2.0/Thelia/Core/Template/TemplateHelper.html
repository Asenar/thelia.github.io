at line 192
number walkDir( string $directory, string $walkMode, Translator $translator, string $currentLocale, string $domain, array $strings)

Recursively examine files in a directory tree, and extract translatable strings.

Returns an array of translatable strings, each item having with the following structure: 'files' an array of file names in which the string appears, 'text' the translatable text 'translation' => the text translation, or an empty string if none available. 'dollar' => true if the translatable text contains a $

Parameters

string $directory the path to the directory to examine
string $walkMode type of file scanning: WALKMODEPHP or WALKMODETEMPLATE
Translator $translator the current translator
string $currentLocale the current locale
string $domain the translation domain (fontoffice, backoffice, module, etc...)
array $strings the list of strings

Return Value

number the total number of translatable texts

Exceptions

InvalidArgumentException if $walkMode contains an invalid value
